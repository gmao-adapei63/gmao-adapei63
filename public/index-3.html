Nouvelle Procédure

Si renseignée (URL ou fichier importé), un bouton "🛂 Exécuter sur le terrain" reste affiché en permanence en haut de cette procédure pour l'ouvrir directement — quel que soit le mode de création utilisé (manuelle, import, JSON, ou lien URL).

⚠️ De nombreux sites (OneDrive, SharePoint...) refusent par sécurité d'être affichés dans une autre application (« framing »). Si le bouton 🛂 n'affiche rien, utilisez l'option B ci-dessous — elle fonctionne toujours.

HTML / Markdown / texte : analysés et convertis automatiquement (titre, sections, listes). JSON déjà au format procédure GMAO : ré-importé directement tel quel. PDF / Word / images / autres fichiers JSON : joints en pièce jointe (le contenu structuré reste à compléter manuellement ci-dessous). Vous pouvez sélectionner plusieurs fichiers à la fois.

Une brève analyse est tentée pour pré-remplir le titre et les étapes si la page est accessible. Si l'analyse automatique est impossible (restriction de sécurité du site distant), l'adresse reste tout de même enregistrée comme page d'exécution terrain, et vous complétez les champs manuellement, exactement comme pour une création classique (catégorie, sécurité, étapes, etc.).
